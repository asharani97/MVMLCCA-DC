# MVMLCCA-DC : Multi-view Multi-label Canonical Correlation Analysis with Dual Correlations 

> **Official implementation** of the paper **MVMLCCA-DC**, a generalization of Canonical Correlation Analysis (CCA) for **cross-modal multimedia retrieval** on **multi-view, multi-label data**, published in the *International Journal of Multimedia Information Retrieval (2026)*.

[![Paper](https://img.shields.io/badge/Paper-Springer-blue)](https://doi.org/10.1007/s13735-025-00384-6)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)
